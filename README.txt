
# Installation 

Cloner le repositorie avec la commande `git clone https://github.com/Flosch37/catchit_back.git`

Installer les dépendances du projet avec la commande `npm install`

Cree une base de données vide avec le nom 'catchit_db' (configurable dans config/config.json)

Executer la commande `npx sequelize db:migrate` afin de lancer les migrations de sequelize
